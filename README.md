Simple and sweet Expense Tracker with backend logic where you can store your data and change any time .

User can see `Total Money` , `Total Income` and `Total Expense` 

## Technology Stack

#### `React` 
#### `Node.js (Express)`
#### `Mongodb (Database)` 



## Available Scripts

### `npm start`

"start": "node server"
It start Node Server in available port
